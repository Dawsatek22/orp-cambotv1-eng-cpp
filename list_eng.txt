Warnings and summaries:
This is a project uses around 5 to 12 volt so be careful.the links for: materials,tools,are there for reference.You do not need to buy them from the linked pages if possible per say.
.if there is made a mistake or some is not made right please notify so i can correct any mistake.
Required skills:
c++ programming,cloning a github repo,using cmake,using vnc viewer to log into the raspberry pi remotely,3d printing or aquiring 3d printed parts,
soldering or someone wh can solder parts,pairing a BLE gamepad to the raspberry pi os bluetooth adapter.

Required software for the raspberry pi:

LCCV: https://github.com/kbarni/LCCV,
cmake: https://cmake.org/download/,
libevdev (you can do a sudo install like this: sudo apt install libevdev2 libevdev-dev -y)
,gcc compiler (is already pre installed on the raspberry pi os)
PiPCA9685: https://github.com/barulicm/PiPCA9685/tree/main.

Required software for the pc/laptop:
vnc viewer: https://www.realvnc.com/en/connect/download/viewer/.
